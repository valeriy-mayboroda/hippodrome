Небольшая игра под названием "Ипподром". Наш ипподром хранит список всех его лошадей
horses типа ArrayList<Horse> в класс Hippodrome.
У каждой лошади на скачках должны быть известны имя (name) и скорость (speed).
Наши лошади будут бежать просто определенное время (100 секунд/"шагов").
Будем определять победителя, как лошадь, пробежавшую наибольшую дистанцию.
Метод move будет управлять движением всех лошадей.
Метод print отрисовывать их на экран.
А метод run - управлять всем этим (это и будет наш забег).
В теле цикла run вызываем сначала move, затем print.
Чтобы весь цикл не отработал за долю секунды - добавляем в него еще Thread.sleep(500).
Забег лошадей отрисовываем точками в консоль.
